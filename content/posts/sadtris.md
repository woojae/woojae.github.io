[Sadtris](https://github.com/woojae/sadtris) is a sad Tetris clone built natively for macOS with SwiftUI. It works pretty well, but the only reason you would use this is if you're on the plane, or in the woods.

Keep it around for your next flight, or when you are backpacking in the woods with a MacBook Pro and no internet. Or don't. It's sad either way.

![Sadtris Screenshot](https://d3nu6st0wla7pm.cloudfront.net/sadtris.png)

## Installation

```bash
git clone https://github.com/woojae/sadtris.git
cd sadtris
swift build -c release
```

The source code is available on [GitHub](https://github.com/woojae/sadtris).
