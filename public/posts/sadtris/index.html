Sadtris

Sadtris is a sad Tetris clone built natively for macOS with SwiftUI. The only reason you would use this is if you’re on the plane, or in the woods.

Keep it around for your next flight, or when you are backpacking in the woods with a giant macbook and no internet. Or don’t. It’s sad either way.

Installation

git clone https://github.com/woojae/sadtris.git
cd sadtris
swift build -c release

The source code is available on GitHub.
